## 🚀 What I'm building at Life2Food

Life2Food is a Colombian platform connecting consumers with stores and restaurants selling quality food near its expiration date at reduced prices.

As COO and technical co-founder, I work across:

- Business strategy and operating model  
- Product development and frontend architecture  
- Merchant operations and partnerships  
- Commerce flows, payments, and platform scaling  
- Process automation and scalable workflows

## 🧩 Life2Food Product Ecosystem

### 📱 Consumer Mobile App (Life2Food-movil)
React Native / Expo application for:
- Product discovery  
- Cart and order flows  
- User profiles  
- Pickup workflows  

### ⚙️ Backend Platform (backendLife2Food)
Spring Boot API with:
- MySQL database  
- Firebase Admin (Auth & services)  
- AWS S3 (storage)  
- Email services  
- Validation and security  
- OpenAPI documentation
